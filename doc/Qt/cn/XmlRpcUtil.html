Qt XML RPC：XmlRpcUtil（简体中文）

Qt XML RPC Qt XML RPC（简体中文）


XmlRpcUtil一般是XML RPC类别在内部处理时使用，使用者一般不应该直接使用XmlRpcUtil 内的功能。

类别宣告


class XmlRpcUtil
{
  public:

    static QString parseTag   (const char    * tag,const QString & xml,int * offset) ;
    static bool    findTag    (const char    * tag,const QString & xml,int * offset) ;
    static QString getNextTag (const QString & xml,int * offset) ;
    static bool    nextTagIs  (const char    * tag,const QString & xml,int * offset)  ;
    static QString xmlEncode  (const QString & raw) ;
    static QString xmlDecode  (const QString & encoded) ;
    static void    log        (int level,const char * fmt,...) ;
    static void    error      (const char * fmt, ...) ;

} ;

static QString parseTag (const char * tag,const QString & xml,int * offset)

解析XML标籤当中的内容。

其中，tag必须为如下的标籤对：

static const char VALUE_TAG  [] = "<value>"  ;
static const char VALUE_ETAG [] = "</value>" ;
两个标籤必须连接在一起。

xml为整个XML文件内容。

offset为目前解析的开始位置。

返回的内容为标籤对当中的内容，offset为整个标籤结束的下一个开始位置。

static bool findTag (const char * tag,const QString & xml,int * offset)

寻找XML标籤内容的起始位置。

tag的标籤对需要如同parseTag一样。

xml为整个XML文件内容。

offset为目前解析的开始位置。

返回的值为是否寻找到标籤对，offset为整个标籤内容的开始位置。

static QString getNextTag (const QString & xml,int * offset)

找到下个XML标籤的位置与标籤名称。

xml为整个XML文件内容。

offset为目前解析的开始位置。

返回的文字字串为标籤名称，offset为整个标籤记号的开始位置。

标籤名称如果长度为0，代表寻找失败，不存在下一个XML标籤。

static bool nextTagIs (const char * tag,const QString & xml,int * offset)

找到下个指定的XML标籤位置与标籤名称。

tag的标籤对需要如同parseTag一样。

xml为整个XML文件内容。

offset为目前解析的开始位置。

返回的值为是否寻找到标籤对，offset为整个标籤内容的开始位置。

static QString xmlEncode (const QString & raw)

将一般文字编码成XML格式。

static QString xmlDecode (const QString & encoded)

将XML格式解码为一般文字格式。

static void log (int level,const char * fmt,...)

这个函式将讯息包装成文字，并且透过XmlRpcLogHandler输 出。

讯息长度的上限是16383。

static void error (const char * fmt, ...)

这个函式将讯息包装成文字，并且透过XmlRpcErrorHandler输 出。

讯息长度的上限是16383。
