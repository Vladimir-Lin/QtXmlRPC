Qt XML RPC：XmlRpcUtil（正體中文）

Qt XML RPC Qt XML RPC（正體中文）


XmlRpcUtil一般是XML RPC類別在內部處理時使用，使用者一般不應該直接使用XmlRpcUtil 內的功能。

類別宣告


class XmlRpcUtil
{
  public:

    static QString parseTag   (const char    * tag,const QString & xml,int * offset) ;
    static bool    findTag    (const char    * tag,const QString & xml,int * offset) ;
    static QString getNextTag (const QString & xml,int * offset) ;
    static bool    nextTagIs  (const char    * tag,const QString & xml,int * offset)  ;
    static QString xmlEncode  (const QString & raw) ;
    static QString xmlDecode  (const QString & encoded) ;
    static void    log        (int level,const char * fmt,...) ;
    static void    error      (const char * fmt, ...) ;

} ;

static QString parseTag (const char * tag,const QString & xml,int * offset)

解析XML標籤當中的內容。

其中，tag必須為如下的標籤對：

static const char VALUE_TAG  [] = "<value>"  ;
static const char VALUE_ETAG [] = "</value>" ;
兩個標籤必須連接在一起。

xml為整個XML文件內容。

offset為目前解析的開始位置。

返回的內容為標籤對當中的內容，offset為整個標籤結束的下一個開始位置。

static bool findTag (const char * tag,const QString & xml,int * offset)

尋找XML標籤內容的起始位置。

tag的標籤對需要如同parseTag一樣。

xml為整個XML文件內容。

offset為目前解析的開始位置。

返回的值為是否尋找到標籤對，offset為整個標籤內容的開始位置。

static QString getNextTag (const QString & xml,int * offset)

找到下個XML標籤的位置與標籤名稱。

xml為整個XML文件內容。

offset為目前解析的開始位置。

返回的文字字串為標籤名稱，offset為整個標籤記號的開始位置。

標籤名稱如果長度為0，代表尋找失敗，不存在下一個XML標籤。

static bool nextTagIs (const char * tag,const QString & xml,int * offset)

找到下個指定的XML標籤位置與標籤名稱。

tag的標籤對需要如同parseTag一樣。

xml為整個XML文件內容。

offset為目前解析的開始位置。

返回的值為是否尋找到標籤對，offset為整個標籤內容的開始位置。

static QString xmlEncode (const QString & raw)

將一般文字編碼成XML格式。

static QString xmlDecode (const QString & encoded)

將XML格式解碼為一般文字格式。

static void log (int level,const char * fmt,...)

這個函式將訊息包裝成文字，並且透過XmlRpcLogHandler輸 出。

訊息長度的上限是16383。

static void error (const char * fmt, ...)

這個函式將訊息包裝成文字，並且透過XmlRpcErrorHandler輸 出。

訊息長度的上限是16383。
